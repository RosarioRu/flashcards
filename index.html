JavaScript(JS)

An object-oriented programming language often used to make web sites interactive.

More info

Operators

Characters such as (+) or (!==) that carry out operations such as math on data, or assigning a value to variables. They can also compare values/data to each other.

More info

Variables

A name given to a value, known or unknown.

More info

Function

A named set of steps/processes that produces a result/output from performing said processes on an inputted value (or values).

More info

Methods

A type of JS function that is called on something, such as on a string.

More info

Arguments

The actual value or values you input/give to a function - the data you want the function/method to perform its steps on. Not to be confused with parameters.

More info

Parameters

A named variable that is assigned to an argument that gets passed into a function. Sort of like a temporary/placeholder for the actual value that gets inputted into a function. Related to, but not the same as an argument.

More info

jQuery

A file/library with pre-written functions for common tasks that you can use in your js files. This allows you to do some things fast, with pre-build functions like alert() without having to write all the code to make them happen.
